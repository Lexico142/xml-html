La función del XML es actuar como una base de datos portátil. Su objetivo es almacenar toda la información de los animes (títulos, géneros, fechas, sinopsis) de forma estructurada, separando completamente el contenido del diseño.

Por otro lado, HTML y CSS se encargan de la presentación visual. El HTML crea la estructura o esqueleto de la página (dónde van los títulos, imágenes y botones), mientras que el CSS aplica el estilo (colores, bordes y maquetación). Gracias a esta separación, el JavaScript puede leer los datos del XML e inyectarlos en el HTML.
